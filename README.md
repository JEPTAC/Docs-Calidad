# Estudio Documental EI V24

Ajustes realizados:

- Paginación obligatoria más estricta antes del pie de página.
- Si el contenido supera el área útil, se fuerza la hoja siguiente.
- Se redujo la capacidad útil estimada de cada hoja para evitar que el contenido invada el pie.
- Se agregó botón "Guardar en navegador".
- El guardado usa localStorage del navegador, no descarga JSON.
- Al abrir nuevamente el archivo en el mismo navegador, carga el último documento guardado.
- No se modificó el sistema de sublistas ni la imagen general restaurada.
